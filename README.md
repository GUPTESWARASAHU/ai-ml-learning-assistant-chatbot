# AI-ML-Learning-Assistant-Chatbot

A Python-based rule-driven learning assistant chatbot that helps students explore technical topics, discover learning resources, and find project ideas.

## Features

* Interactive text-based chatbot
* Topic explanations for:

  * Python
  * Artificial Intelligence (AI)
  * Machine Learning
  * Data Science
  * Cyber Security
  * Data Structures
  * Prompt Engineering
  * Sports
* Learning roadmaps and key skills
* Hindi YouTube learning resources
* Google Search integration
* GeeksforGeeks resources
* Kaggle learning resources
* freeCodeCamp learning resources
* Minor project recommendations
* Major project recommendations
* Browser search support
* Case-insensitive user input

## Concepts Used

* Python Dictionaries
* Functions
* Loops
* Conditional Statements (if-elif-else)
* Lists
* Strings
* User Input/Output
* Webbrowser Module

## Complete Flow

Start
↓
Display Welcome Message
↓
User Enters Input
↓
Input = "topics" ?
├─ Yes → Show Available Topics
└─ No
↓
Topic Exists ?
├─ No → Show Error Message
└─ Yes
↓
Display Topic Information
↓
Show Skills Required
↓
Show Learning Resources
↓
Show Minor Projects
↓
Show Major Projects
↓
Ask to Open Google Search
├─ Yes → Open Browser
└─ No → Continue
↓
User Enters New Topic
↓
Input = "bye" ?
├─ Yes → Exit Chatbot
└─ No → Continue Conversation
